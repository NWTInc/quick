Quick Start Instructions
  1. Pair to the Smart Socket via Android → Settings → Bluetooth → Scan
  2. Enter Bluetooth pins 1111 or 1234 (factory default)
  3. Start RAD Connect - PRODOCS
  4. The paired Smart Socket will show in the list of Devices
  5. Tap on the paired Smart Socket
  6. Set your Smart Socket to Peak Mode by repeatedly press Right Arrow
